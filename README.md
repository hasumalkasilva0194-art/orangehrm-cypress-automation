# Cypress Automation Testing Projects

## 📌 Project Overview
This repository contains automation testing projects developed using Cypress and JavaScript.

The projects include:
- OrangeHRM Automation Testing
- SauceDemo Automation Testing

These projects were created to practice real-world QA automation concepts including UI validation, login functionality, cart functionality, and form validation testing.

---

# 🚀 Projects Included

## 1️⃣ OrangeHRM Automation Testing

### Features Tested
- Login validation
- Invalid credential handling
- Empty field validation
- Forgot Password functionality

### Test Cases
- TC_01 to TC_13

## 2️⃣ SauceDemo Automation Testing

### Features Tested
- Login functionality
- Add to cart
- Remove from cart
- Cart badge validation
- Checkout flow
- UI validations

## ▶️ Run the Project

### Install dependencies
```bash
npm install
```

### Open Cypress
```bash
npx cypress open
```

### Run all tests
```bash
npx cypress run
```
